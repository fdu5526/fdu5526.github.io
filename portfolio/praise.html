PRAISE

What Is It?

PRAISE allows art students with hearing disabilities to peer review each other's works with an iPad app. Students can write reviews, rate, and annotate aspects of drawings. It is my research project for CMU's Human Computer Interaction Institute that I started in January 2013.

What Is It Written In?

The iPad app is written in Objective-C. The backend server is written in Python, and the frontend of the server for teachers is written in Actionscript and MXML.

What Did I Do?

I first designed the user interface of the app, starting from brainstorming to paper prototyping and finalization using an iterative process. Then I implemented the finalized design. At the same time, I improved the backend of the app and the server it uses to be more reliable and robust with more features.

What Did I Learn?

I learned user interface design processes such as paper prototyping, iOS development and communication between programs via network. I also learned how to use Mac OS X, Xcode, and Flash Builder.
