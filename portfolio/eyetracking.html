Eye Tracking

Know Where People Look During Usability Testing

What Is It?

Eye Tracking is my intern project at Epic in the summer of 2014. The software has 3 parts: it lets designers create eyetracking usability tests, conducts them to collect data, and analyzes them via visualizations. The purpose of this project is to be an alternative to the $30,000 Tobii suite. We are using another device called The Eye Tribe which only costs $100.

How Is It Made?

We started off with only The Eye Tribe's API reference, and built up the entire application from scratch. We used C# for the bulk of our application, WPF for the frontend, and SQL for data storage in our backend.
